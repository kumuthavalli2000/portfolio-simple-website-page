# Personal Portfolio Website

A simple personal portfolio website built using **HTML5** and **CSS3**. The website consists of two pages: a **Home** page that introduces me and highlights my profile, and a **Contact** page where visitors can find my contact information.

## Features

* Clean and minimal design
* Home page with personal introduction
* Dedicated Contact page
* Easy navigation between pages
* Built with semantic HTML and basic CSS

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was created to strengthen my understanding of HTML and CSS by building a basic multi-page portfolio website. It serves as a foundation for learning more advanced front-end development concepts.

* Include Projects and Skills sections
